三角关系

发表于 2018-12-26

川农牛8：三角关系

题目描述：

给你两个数 N 和 K

对于一个三元组（a，b，c）满足 a+b，b+c，c+a 都可以被 K 整除。

N，K，a，b，c 都是正整数，a，b， c 都不大于 N ，求满足条件的三元组个数。

Input

第一行两个整数 N 和 K

  • 1≤ N,K ≤2 x 10^5

Output

  • 输出满足条件的三元组个数

定义：两个三元组 (a,b,c),(d,e,f)(a,b,c),(d,e,f) 相同，当且仅当 a=d,b=e,c=fa=d,b=e,c=f。

限制条件：Time Limit 1000 MS，Memory Limit 256 MB

题目分析

看到此题时间限制1秒内，想要通过三重循环做肯定时不可能的了，当然要是三重循环就能做出来，那么这道题也没有什么价值，也不会出现在这里了

我们需要的是分析a，b，c，以及 K 它们内在的关联

当 K 为奇数时，a，b，c 只能为 K 的倍数才满足条件。(a+b) % K = (b+c) % K = (a+c).可得 (2a) % K =0，或者 (2c) % K =0。因为 K 为奇数，所以a,b,c*只能为 K 的倍数才满足。

当 K 为偶数时，a，b，c 为 K 的倍数以及等于 K/2 时也满足。同上，但是 K 为偶数时，a,b,c可为 K/2。

因此有如下代码：


import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        long N, K, ans;long T=0;
        N = sc.nextInt();
        K = sc.nextInt();
        long ans1 = N / K;
        if (K/2==N){
            System.out.println(1);
            return;
        }
        if(N<K)
        {
            System.out.println(0);
            return;
        }
        ans = ans1*ans1*ans1;
        if (K % 2 == 0) {
            T=(N-K/2)/K+1;
            ans+=Math.pow(T,3);
        }
        System.out.println(ans);
    }
}

我的第一篇博客

开心

此刻先让我表达一下心中的兴奋之感吧，从周五晚上开始搭建这个博客，遇到各种各样的问题，周六晚上终于搞定搭建完成并可以部署到GitHub上面了，今天上午开始了解Markdown语法和找之后博客中的图床，便于之后的博客贴图进去，也好管理，之前有看到人推荐用七牛云，今天网上看了半天，发现用了一个月后就会回收测试域名，之后我们只能用自定义域名，也就是我们还必须去买一个域名，虽然域名也不贵，但是还要备案，感觉挺麻烦的，然后突然想起我还有腾讯云呢，送的50G对象存储还没有用，于是决定用腾讯云的对象存储作为我的图床了，刚刚开始用Hexo+GitHub+node.js+git来搭建的博客，感觉还有挺多不懂的，只能慢慢看文档网上查来学习怎么用吧，花了我一天多时间，我心心念的博客平台终于搭建好了，就让我再开心一下吧，哈哈！

为什么想开这个博客

开心完了，说说为什么就想来开这么一个博客吧，其实在去年寒假考试完我就花了一个下午来搭建这个博客，当时教程也找得比较全，现在还记得里面都是一步一步该做什么讲得很好的，不过由于当时对于这种搭建博客方式不是很理解，说白了就是菜，对于GitHub，感觉上面都是英语，限制了我前进的脚步，看着就累，现在想想真的好傻。它的一些功能也不懂，我就只能按照教程的步骤来，后来可能版本还是什么的遇到了一点问题，于是就不知道怎么解决了，真心觉得不会解决问题才是最痛苦的，就搁置了，后来电脑也出了点小问题，系统被重装了，什么环境都没有了就更没想起这回事了，周五突然看到一个博客，感觉超好看的，就想着这周末一定要搭一个出来，就折腾了一天
。受我加入的一个大佬圈子的影响，让我知道了 专业课，英语，写作 三大技能是非常重要的，所以写博客是一直的想法，可惜拖了很久，今天终于开始了，希望不会晚，现在的互联网时代，我们每天都被各种各样的信息包围着，我们也更加方面从网上获取知识，可惜很多都是看时爽，看了之后大脑还是没有任何的思考以及收获，像平时逛的 知乎，微信的公众号等 里面确实有很多有用信息，也有很多大佬分享各种人生经验，我们都知道人的成功是不可复制的，但是他们分享出来的经验是可以借鉴的，让我们少走弯路。我就是想把平时看到的信息感悟，体会，收获给记载下来，也算作为记录自己生活的一种方式吧！

目标及自我要求

感觉都这么多字，这个小节应该放到下一篇博客的，但是想起目标和要求 还是放到第一篇博客里面比较好，也能更好的对自己一个鞭策，以前高中班主任陈老师说过一句话

我们不是看制度建立得多么完善，而是看执行的结果怎么样

既然自己给自己设定了目标和要求那就必须去完成，这也是搭建这个博客的一个初衷

  1. 每周一篇非技术型文章（主要说一周来自己的所见所闻所感以及收获）类似周记
  2. 每周至少一篇技术性文章（可关于平时做的算法题的题解，以及项目中学到的一些新技术）

后记

第一次写这种纯文字的博客，花了一个多小时，重要的还是之后一定要坚持下去吧，再加一句我之前放在 QQ 签名上的一句话吧

没有任何成就比用时间堆砌起来的成就更难让人难以超越！
