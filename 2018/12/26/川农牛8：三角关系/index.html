川农牛8：三角关系

题目描述：

给你两个数 N 和 K

对于一个三元组（a，b，c）满足 a+b，b+c，c+a 都可以被 K 整除。

N，K，a，b，c 都是正整数，a，b， c 都不大于 N ，求满足条件的三元组个数。

Input

第一行两个整数 N 和 K

  • 1≤ N,K ≤2 x 10^5

Output

  • 输出满足条件的三元组个数

定义：两个三元组 (a,b,c),(d,e,f)(a,b,c),(d,e,f) 相同，当且仅当 a=d,b=e,c=fa=d,b=e,c=f。

限制条件：Time Limit 1000 MS，Memory Limit 256 MB

题目分析

看到此题时间限制1秒内，想要通过三重循环做肯定时不可能的了，当然要是三重循环就能做出来，那么这道题也没有什么价值，也不会出现在这里了

我们需要的是分析a，b，c，以及 K 它们内在的关联

当 K 为奇数时，a，b，c 只能为 K 的倍数才满足条件。(a+b) % K = (b+c) % K = (a+c).可得 (2a) % K =0，或者 (2c) % K =0。因为 K 为奇数，所以a,b,c*只能为 K 的倍数才满足。

当 K 为偶数时，a，b，c 为 K 的倍数以及等于 K/2 时也满足。同上，但是 K 为偶数时，a,b,c可为 K/2。

因此有如下代码：


import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        long N, K, ans;long T=0;
        N = sc.nextInt();
        K = sc.nextInt();
        long ans1 = N / K;
        if (K/2==N){
            System.out.println(1);
            return;
        }
        if(N<K)
        {
            System.out.println(0);
            return;
        }
        ans = ans1*ans1*ans1;
        if (K % 2 == 0) {
            T=(N-K/2)/K+1;
            ans+=Math.pow(T,3);
        }
        System.out.println(ans);
    }
}
